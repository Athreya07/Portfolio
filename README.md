# Atharva VS Code Portfolio

A premium Software Engineer portfolio inspired by VS Code, GitHub Desktop, and GitHub Dark Mode.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- React Icons

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize

Edit your personal content in:

```txt
src/data/portfolio.ts
```

Add your resume here:

```txt
public/resume.pdf
```

## Deploy

Deploy easily on Vercel or Netlify.
